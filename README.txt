MidpSSH is an SSH and Telnet client for MIDP 1.0 / 2.0
devices such as Java-capable cellphones.

MidpSSH is based upon Floyd SSH and Telnet Floyd by
Radek Polak (http://phoenix.inf.upol.cz/~polakr/).

This includes sources from Matthias L. Jugel,
Marcus Meiﬂner (javassh.org) and also the
GNU Classpath project.

MidpSSH has been enhanced by Karl von Randow from Floyd SSH.
Including:
* integration of Telnet and SSH into the one midlet;
* user interface development;
* refactoring and bug fixes;
